# dotfiles

## Installation

### From Scratch

1. Install XCode from the app store
1. Run `xcode-select --install`
1. Clone this repo: `git clone https://github.com/jetpackjarrett/dotfiles.git ~/.dotfiles`
1. Run `sh ~/.dotfiles/install.sh`

### Locally (Already cloned)

```bash
sh install.sh
```
